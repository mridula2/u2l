================
STKHL 1.0 の説明
================
2015/02/03 

- アーカイブ名は、STKHL-1_0.tar.gz です。

- 動作環境 : 
  STKHL 1.0 は、HP-UX 11i 上で操作します。
  ( Linux 上では動作しません )

- インストール用アーカイブファイルは、etc/opt/STKHL と opt/STKHL を相対パスで
  アーカイブしたものです。インストールするためには、root ディレクトリから、
  アーカーブを "root" 権限で展開してください。

- STKHL 1.0 の制限
  STKHL 1.0 は、テキストモードのみが使用できます。
  HTML モードで出力される HTML ファイルの内容や、TEXT モードで出力される出力の
  HTML のリンク先の内容は正しいものではありませんので利用しないでください。

------------------------------------------------------
●データの管理
lib/db のデータは、CVS 上で管理しています。

CVSROOT は hpjctc2(16.146.89.146):/data/cvsrepo で、
トップディレクトリは、STKHL-data です。

STKHL 1.0 には、"STKHL-1_0" のタグが付けてあります。

  $ mkdir work
  $ cd work
  $ cvs co -r STKHL-1_0 STKHL-data

この中で、管理しているものは、STKHL の lib/db の内容です。

ファイルの生成は、gen-stkhl.sh を使用します。

データの管理の詳細は、『STK データ管理手順書』に記述があります。


補足: CVS 上のSTKHL-dataと同じディレクトリに STKHL-cmd というディレクトリも
      あります。これは、STKHL-1_0.tar.gz と同等な内容をチェックインしたものですが、
      特に CSV 上で管理することも無いので、今後は使用しない方針です。
