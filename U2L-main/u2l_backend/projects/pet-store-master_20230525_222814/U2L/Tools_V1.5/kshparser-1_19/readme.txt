●使用方法

　hpscanツールをインストール後、/usr/local/hpscan/shanalyze/に
　kshparserとしてコピー、またはシンボリックリンクする

●ソースコード管理
　CVSROOT は hpjctc2(16.146.89.146):/data/cvsrepo

  $ mkdir work
  $ cd work
  $ cvs co -r kshparser-1_19 kshparser
  $ cd kshparser
  $ ./make.sh
  $ mv kshparser kshparser-1_19.xxx
